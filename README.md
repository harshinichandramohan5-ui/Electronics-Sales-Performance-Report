# Electronics-Sales-Performance-Report
Sales analysis workbook: 700 orders across 5 laptop models (Acer Aspire 3, Dell Inspiron 15, Lenovo IdeaPad Slim 3i, Mi NoteBook Ultra, Samsung Galaxy Go), 4 Indian regions/states. Includes pivot tables by product, region, state &amp; month (Jan–Jun 2020), a dashboard, and raw transaction data. Total: ₹34,759,688, 3,762 units sold.
