# 📊 Sales Analysis Workbook

## 📌 Project Overview
The **Sales Analysis Workbook** is an interactive Microsoft Excel project designed to analyze laptop sales performance using Pivot Tables, Pivot Charts, and an interactive dashboard. The workbook contains 700 sales orders across five laptop models sold in different regions and states of India. It helps users explore sales trends, product performance, regional distribution, and monthly revenue through dynamic visualizations.

---

## 🎯 Project Objectives
- Analyze overall sales performance.
- Compare sales across laptop models.
- Evaluate regional and state-wise sales.
- Track monthly sales trends from January to June 2020.
- Provide an interactive dashboard for business insights.

---

## 📂 Dataset Summary
- **Total Orders:** 700
- **Total Units Sold:** 3,762
- **Total Sales:** ₹34,759,688
- **Time Period:** January – June 2020
- **Laptop Models:**
  - Acer Aspire 3
  - Dell Inspiron 15
  - Lenovo IdeaPad Slim 3i
  - Mi NoteBook Ultra
  - Samsung Galaxy Go
- **Regions Covered:** North, South, East, and West India
- **Analysis Level:** Product, Region, State, and Month

## ⚙️ How to Use
1. Download the Excel workbook.
2. Open it in Microsoft Excel 2019 or Microsoft 365.
3. Enable editing if prompted.
4. Click **Data → Refresh All** to update Pivot Tables if needed.
5. Use the dashboard filters and slicers to explore different sales perspectives.
